引脚设置
================================================================================
PB10：SCL
PB11：SDA 


学习目的
================================================================================
以软件模拟IIC时序方式与从机（SHT30）通信，了解I2C通信的关键操作步骤，以及对读写时序的控制


测试环境
================================================================================

测试用板:
STM32F103C6T6
---------------------

辅助工具:
---------------------

辅助软件:
---------------------
串口调试助手
================================================================================
使用步骤
1、PB10、PB11引脚与SHT30的SCL/SDA正确连接，温湿度传感器SHT30的供电为3.3V
2、重新编译并运行
3、在调试窗口观察打印信息，如果与实际一致，运行正常

================================================================================

================================================================================
